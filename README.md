# 域名证书监控
用于获取监控域名证书的有效期

## 功能特点

- 监控多个域名的 HTTPS 证书有效期
- 自动检查证书状态并更新信息
- 支持添加、编辑和删除域名
- 定时任务自动检查所有域名证书

# 启动方式

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
